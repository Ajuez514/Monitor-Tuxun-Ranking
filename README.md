# 图寻 (Tuxun.fun) 排行榜监控脚本 (MonitorRank)

这是一个用于监控图寻网站 (tuxun.fun) 排行榜用户积分变化的 Python 脚本。它能够实时追踪世界和中国排行榜，并在玩家分数、排名发生变动时，在控制台输出详细的对局信息，帮助你第一时间了解顶尖玩家的动态。

---

## ✨ 功能特性

-   **实时监控**：可选择监控世界、中国或两个排行榜。
-   **变动捕捉**：自动检测玩家积分变化、新上榜或掉出榜单。
-   **丰富信息**：输出包含时间、模式、排名、玩家、分数变化、连胜/连败、胜率和对局时长的详细通知。
-   **彩色高亮**：使用多种颜色区分不同信息（如分数增减、胜率高低、玩家排名），让关键数据一目了然。
-   **终端交互**：内置指令系统，可在程序运行时动态调整监控频率、模式和时区。
-   **玩家查询**：可随时查询指定UID玩家的详细档案和最近战绩。
-   **智能重连**：在网络连接失败时自动尝试重连，避免程序中断。
-   **超链接支持**：在现代终端中，可以直接点击玩家UID、复盘链接跳转到浏览器。



## 🚀 安装与使用

1.  前往本项目的 [GitHub Releases](https://github.com/Ajuez514/Monitor-Tuxun-Ranking/releases) 页面。
2.  下载最新的 `Monitor-Tuxun-Ranking.exe` 文件。
3.  双击运行即可！



## ⌨️ 指令说明

程序运行时，你可以在控制台输入以下指令并回车：

-   `speed <5|10|60>`：设置监控频率（5秒、10秒或60秒一次），默认为10秒一次。
-   `type <world|china|both>`：设置要监控的排行榜类型。
-   `time <时区偏移>`：设置显示时间的时区（例如，东八区输入 `8`，西五区输入 `-5`），默认为东八区。
-   `show <uid>`：显示指定UID玩家的详细档案和最近对局信息。
-   `stop` / `continue`：暂停 / 继续监控。
-   `help`：显示帮助信息。
-   `q` / `quit` / `exit`：退出程序。
